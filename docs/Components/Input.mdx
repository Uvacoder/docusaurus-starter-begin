```jsx
<Input />;
```

## Props

| Prop        | Accepts                  | Type   | Default  | Description                                         |
| ----------- | ------------------------ | ------ | -------- | --------------------------------------------------- |
| placeholder | string                   | string |          | Placeholder for the Input component                 |
| value       | string                   | string |          | The default value for the Input component           |
| onChange    | event                    | event  |          | The `onChange` event related to the Input component |
| size        | `small` `medium` `large` | string | `meduim` | The size of the Input component                     |
| className   | string                   | string |          | A list of external classNames for the component     |
